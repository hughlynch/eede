# eede

**An open source Earth Engine development environment.**

Pronounced "Edie." See [VISION.md](VISION.md) for the full design document.

## What is this?

eede is a proposal and prototype for a modern, open source development environment for Google Earth Engine. It aims to provide a migration path for the ~500,000 EE developers currently locked into the aging Code Editor, while adding capabilities the community has been asking for: dual-language support, cell-based notebooks, real debugging, a modern batch system, and an extensibility framework.

## Status

**Pre-prototype.** This repo contains the vision document, architecture sketch, and early prototype code. We're sharing it to gather feedback from the EE community before building further.

## Why?

The Earth Engine Code Editor has served the community well for over a decade, but it hasn't kept pace with modern development practices. Meanwhile, EE's Python API is growing fast but has no equivalent IDE experience. The community deserves better tooling — and the EE API is open source and fully functional outside the Code Editor, so building it is entirely feasible.
